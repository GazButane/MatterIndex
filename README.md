# Matter index

Material manager for 3D software (Blender, substance Painter, Cinema 4D, etc, ...) 


## Features

- Search by tags
- Search bar
- Easily copy objects
- Open source


## Installation note

 Add a folder named **"things"** at the root of the software and drag your **unzipped** materials into it.
